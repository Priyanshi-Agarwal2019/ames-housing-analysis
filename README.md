# Ames Housing Price Analysis

This project analyzes the factors affecting house prices using the
Ames Housing dataset (1,460 observations, 79 variables).

## Key Components
- Data preprocessing and missing value handling
- Exploratory data analysis with interpretation
- Log-transformed linear regression modeling
- Model evaluation using MAE, RMSE, and MAPE
- Residual diagnostics and neighborhood analysis

## Tools Used
R, tidyverse, car, ggplot2, caret,e1071,skimr

This project was completed as part of the MTTS Statistics, AI & ML program.
